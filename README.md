Derroteros de la navegación – Primera vuelta al mundo

Este proyecto web es una página sobre los derroteros de la navegación durante la primera vuelta al mundo (1519–1522), protagonizada por Juan Sebastián Elcano tras la expedición iniciada por Fernando de Magallanes.
